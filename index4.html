Ferramentas Utilizadas

Ferramentas de desenvolvimento são programas utilizados por desenvolvedores para criar, executar, depurar (debug) e testar seus aplicativos/softwares. Algumas delas são de uso universal, como no caso do editor de código-fonte e do compilador – as quais costumam ser combinadas na realização das tarefas. Essas ferramentas podem ser executadas separadamente – geralmente a partir da linha de comando – ou, mais frequentemente, integrarem um único grande programa, conhecido como Ambiente de Desenvolvimento Integrado (IDE). Editor de código-fonte

O editor de código é uma das ferramentas mais importantes para os desenvolvedores. Ele funciona, basicamente, como um editor de texto, porém também é projetado para gerenciar e editar o código-fonte de programas e aplicativos. Além disso, geralmente conta com recursos integrados e mais avançados em comparação ao editor de texto tradicional – tais como o realce de sintaxe, o preenchimento automático de trechos de código, entre outras funcionalidades. Há diferentes tipos de editores de código no mercado, desde versões com suporte específico para determinada linguagem de programação até aqueles que suportam as mais diferentes linguagens. Entre os editores mais usados estão o Visual Studio Code (VSCode), Atom, Notepad++ e Brackets.
